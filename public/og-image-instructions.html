Creating Your FreightFox Open Graph Image

This guide will help you create an optimal Open Graph image for FreightFox that will look good when shared on social media platforms.

Optimal Specifications

  • Format: PNG or JPG
  • Dimensions: 1200 × 630 pixels (2:1 aspect ratio)
  • File size: Less than 1MB (ideally under 300KB)
  • Important content: Keep main content in the center 1080 × 600 pixels as some platforms may crop edges

Creation Steps

  1. Open a graphics editor like Adobe Photoshop, GIMP, Figma, or Canva
  2. Create a new document with 1200 × 630 pixel dimensions
  3. Set up a background that matches FreightFox branding (gradient from #FF6E33 to #99421F or use a solid color)
  4. Place the FreightFox logo (from src/assets/Logo.svg) prominently in the center
  5. Add a tagline or brief description if desired (keep it minimal)
  6. Export as PNG with the name "og-image.png"
  7. Replace the placeholder file in the public folder

Tips for Better Engagement

  • Use your brand colors consistently
  • Make sure any text is legible (not too small)
  • Include your logo
  • Keep the design simple and focused
  • Test how it looks when shared on different platforms

Example Code for Creating Canvas-based Image

You could also generate the image programmatically using HTML5 Canvas:

// Create a canvas element
const canvas = document.createElement('canvas');
canvas.width = 1200;
canvas.height = 630;
const ctx = canvas.getContext('2d');

// Create gradient background
const gradient = ctx.createLinearGradient(0, 0, 0, 630);
gradient.addColorStop(0, '#FF6E33');
gradient.addColorStop(1, '#99421F');
ctx.fillStyle = gradient;
ctx.fillRect(0, 0, 1200, 630);

// Load and draw logo
const logo = new Image();
logo.onload = function() {
  // Center the logo
  const logoWidth = 500; // Adjust as needed
  const logoHeight = 100; // Adjust as needed
  ctx.drawImage(logo, (1200 - logoWidth) / 2, (630 - logoHeight) / 2, logoWidth, logoHeight);
  
  // Add tagline
  ctx.font = 'bold 32px Montserrat';
  ctx.fillStyle = 'white';
  ctx.textAlign = 'center';
  ctx.fillText('Smart Freight Management', 600, 430);
  
  // Convert to PNG
  const dataUrl = canvas.toDataURL('image/png');
  
  // You can then download this or save it to your server
};
logo.src = '/src/assets/Logo.svg';

Once you have created your og-image.png file, place it in the public folder to complete your Open Graph setup.
